		Installation Process
		####################

	1. open the "setup.bat"

	2. run "SkyRAT" as administrator

	3. Close "SkyRAT" when you see an flashing error (Yes we know...)

	4. Run "SkyRAT"

		   # IMPORTANT! #
	   If the SkyRAT Folder is moved
	 anywhere else, please repeat these
	steps again to make sure everything
	        is working properly!!!
